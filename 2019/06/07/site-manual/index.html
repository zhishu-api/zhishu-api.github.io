  • @[css selector]@[attr]:[reg] 表达式

    • @ 是分隔符, 可用其他字符代替

    • [css selector] 是标准的css选择器, 请参考下面的文档

    • [attr]:[reg] 可为空, 默认获取 text

      • [attr] 是需要获取的属性. 例如: src href text; 可以为空, 默认获取 text
      • [reg] 是正则表达式. 获取 [attr] 值后进行正则运算
